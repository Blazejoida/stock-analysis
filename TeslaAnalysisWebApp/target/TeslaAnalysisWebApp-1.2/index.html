Tesla Stock Analysis Application

Perform Analysis

Analysis Types Description:

  • Highest Volume Day: Finds the day with the highest trading volume
  • Highest Volume Year: Finds the year with the highest total trading volume
  • Price Sorting: Sorts stock data by closing price in ascending order
  • Price-Volume Correlation: Calculates Pearson correlation between opening price and volume

Navigation:

Instructions:

  1. Enter file path: Type the full path to your CSV file containing Tesla stock data in the "File Path" field (e.g., C:/data/tesla_stock.csv)
  2. Select analysis type: Choose one of the four available analysis types from the dropdown menu
  3. Submit: Click "Perform Analysis" to execute the analysis and save results to database
  4. View history: Click "View Analysis History" to see all past analyses from database and cookie information

What you'll see:

  • [FROM COOKIE] - Information stored in your browser cookies (session data, analysis count)
  • [FROM DATABASE] - Information stored permanently in the database (all analysis operations)
  • [IN MEMORY] - Information stored temporarily in application memory (current loaded data)

Note: The CSV file should contain columns: Date, Open, High, Low, Close, Adj Close, Volume
